Submissão de Questões

Para uma submissão ser aceita ela deve seguir o template XML abaixo.

Para mais informações sobre o padrão XML consulte esse link: W3 Schools.

<?xml version="1.0" encoding="UTF-8"?>
<question id="1" level="0" subject="graphs">
   <text>o texto da questão.</text>
   <explanation>a explicação.</explanation>
   <choice type="correct">
      <text>primeira possivel resposta</text>
      <explanation>explicacao da resposta</explanation>
   </choice>
   <choice type="incorrect">
      <text>segunda possivel resposta</text>
      <explanation>segunda explicacao da resposta</explanation>
   </choice>
</question>

Você pode fazer o upload de várias imagens para melhor compreensão da questão. Tags html são reconhecidas e aconselhadas para melhor experiência na resolução da questão. Links para aprendizado são desejáveis.
